What's new?

wpStickies 2.0 is our new major update, we are continuing to evolve our every items with the latest expectations. In this new version we did countless improvements, some of them are really big and radical.

  • Brand new stunning admin user interface We are continuing to update our items with our brand new and beautiful user interface. It is easier to use and it gives you fast access to any settings you may search. And it is really sexy.
  • Localization ready The new version is localization ready, so you can translate the entire plugin, including the admin user interface.
  • Multisite ready wpStickies 2.0 is multisite ready, so it supports network-wide activation and automatically handles new sites in your network. Also, super admins now have full access across every network site.
  • Auto-updates The new version has auto-update feature, so you can receive new versions with all the new features on-the-fly.
  • Improved responsiveness We did several improvements on responsiveness and it is now more reliable and entirely dynamic.
  • Shortcode support You can now use shortcodes inside stickies and the plugin automatically parses them and show the corresponding content.
  • Better documentation with contextual help menus If you read this, you can see that we moved our entire documentation and integrated here with this contextual help menu, so you can access it quickly at any time. We also rewrote the entire documentation, now it is easier to understand and offers more information.
  • Countless other improvements There are several other improvements in this version such as relative URLs for moving your site, link a wpStickies enabled image, the preview now doesn't requires enabled rewrite engine, jQuery 1.9 compatibility, option to delete a sticky permanently and many more.
