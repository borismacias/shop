Language support

wpStickies is translation and localization ready. If you want to translate the entire plugin to your own language including the admin area, you can find all the details in the documentation that came with the package that you downloaded from CodeCanyon. For translating the plugin on your front-end, see the section below.

Translating wpStickies

wpStickies works together with translation plugins like qTranslate seamlessly. In the corresponding text fileds (see the "Language" main tab) you can use the shortcode of your translation plugin.

qTranslate example

qTranslate a widely known plugin for translate your WP content. Below there is an example to how to use qTranslate if you choose this plugin for your site.

Example: [:pt]portuguese text[:en]english text
